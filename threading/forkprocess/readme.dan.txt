Linux project to study the threading programming in C. This project focus on the fork function, which create a duplicate of the calling thread.

CMake Building project steps
=============================

From a Terminal
1 - mkdir build  (if no build folder already existing)
2 - cd build
3 - cmake ..   (will create all the file required to compile projec)
4 - make

CodeBlocks
===========

'codeblocks' contains already a Codeblocks project file to edit and compile
